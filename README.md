Hi in this project we have to make a simple website using only HTML

To make a website we need a boilerplate which is essential for every website.

after that we need to make use the <body> tag which displays all the content on

the webpage.

Next we make a heading using the <h1> tag as instructed named Recipes. This is 

the homepage so index.html file.

Then we make links to the recipies using the <a> tag and some attributes like 

target and rel.

then we make boilerplates for each website connected by the link and show an 

image using the <img> tag and rcs attribute.

Next we write a description with the <p> tag.

then we put a headings for ingredients and steps then make lists using the <ul> 

tag for un ordered lists and the <ol> tag for ordered lists.

after that we fill out the steps using <li> tags. 

then we put a link to the main webpage.

after everything we save the files and add these files to our new repository 

by using $ git add "file name" 

then we commit these files to the repost using $git commit -m"Add (file name)"

after that we push the files to git using $git push origin main.
